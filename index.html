Student Info CLI App

This was a project from the first semester. The purpose of the project was to test whether the students understood the concepts of object oriented programming and classes. It is a simple console app which lets the user input the name, age, and address of a "student" object which will be recorded and stored. The user can then also edit an existing record, or display the stored information.

Ice Cream Stock CLI App

Another project from the first semester of my first year. The purpose of this assignment was to test if the student understood how to manage and manipulate arrays, strings, and loops. This console based application allows the user to input a number of details for the stock of an ice cream store, up to 15 buckets. The user also has options to edit existing details, delete existing details, and list all existing details.
Here is the project on Github

Simple Storefront Website

Here is the project on Github

Airline Seat Reservation App

This project was at the start of the second semester of year one, and it was supposed to test the students' understanding of multi-dimensional arrays and windows form controls, as well as make sure the students could interact with the windows form controls well. The application allows a user to book names into seats in a theoretical airplane, or enter a waiting list if all seats are full. The user can also cancel a seat that has already been booked and automatically give the seat to the first name on the waiting list.
Here is the project on Github
